# Data Science Tech Stack - Day 4

## Python Revision

### Topics Revised

- **Lists**

  - Properties: Ordered, mutable
  - Common methods: `append()`, `extend()`, `insert()`, `pop()`, `remove()`, `sort()`, `reverse()`

- **Tuples**

  - Immutable, ordered
  - Methods: `count()`, `index()`

- **List Comprehension**

  - Compact way to create lists
  - Example: `[x**2 for x in range(5)]`

- **Sets**
  - Properties: Unordered, unique elements
  - Methods: `add()`, `remove()`, `union()`, `intersection()`, `difference()`

## Key Learnings

- Difference between **mutable (list, set)** and **immutable (tuple)** structures.
- Practiced **list comprehension** for concise coding.
- Revised key **set operations** for data cleaning and uniqueness.
